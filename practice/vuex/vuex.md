# vuex

> 安装：

    npm install vuex --save

Vuex 是一个专门为 Vue.js 应用设计的 `状态管理模型 + 库`。它为应用内的所有组件提供集中式存储服务，其中的规则确保状态只能按预期方式变更。是一个非常简单的“单向数据流”思想的体现

* state (状态)，驱动我们应用的真实的源；
* view (视图)，对应着 状态 的声明式映射；
* actions (动作)，用户在 视图 上的输入引起状态的更改的可能方式。
